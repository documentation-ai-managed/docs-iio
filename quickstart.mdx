## Prerequisites

Before starting, ensure you have:

<Callout kind="info">

- A valid email address for account creation
- Browser support: Chrome `<58`, Firefox, or Safari `<15`
- Team member emails ready for invitations

</Callout>

## Create Your iio Account

Sign up in under 2 minutes.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, name, and password. Verify via the email link sent to you.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Select your workspace type (Personal or Team) and customize your profile.
  </Step>
</Steps>

## Invite Team Members

Collaborate seamlessly by adding your team.

<Tabs>
  <Tab title="Email Invite" icon="mail">
    <Steps>
      <Step title="Open Settings" icon="settings">
        Click your profile icon > **Team Settings**.
      </Step>
      <Step title="Add Members" icon="users">
        Enter emails, select roles (`admin`, `editor`, `viewer`), and send invites.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Bulk Import" icon="upload">
    Download the CSV template, fill with emails and roles, then upload.

    Example CSV format:

    ````csv
    email,role
    team@company.com,editor
    dev@company.com,admin
    ````
  </Tab>
</Tabs>

<Callout kind="tip">

Invites expire after 7 days. Resend if needed.

</Callout>

## Set Up Your First Space

Organize your project documentation.

<Steps>
  <Step title="Create New Space" icon="plus">
    From the dashboard, click **New Space** and name it (e.g., "Project Docs").
  </Step>
  <Step title="Configure Basics" icon="settings">
    Set visibility (`public`, `private`), add a custom domain if desired, and choose a template.
  </Step>
  <Step title="Add First Document" icon="file-text">
    Click **New Doc**, title it "Introduction", and start editing with Markdown or rich text.
  </Step>
</Steps>

## Integrate via API (Optional Quick Example)

For automation, use the Spaces API.

<CodeGroup tabs="JavaScript,cURL">
  ````javascript
  const response = await fetch('https://api.example.com/v1/spaces', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer YOUR_API_KEY`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'My New Space',
      visibility: 'private'
    })
  });
  const space = await response.json();
  console.log(space.id);
  ````
  ````bash
  curl -X POST https://api.example.com/v1/spaces \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "name": "My New Space",
      "visibility": "private"
    }'
  ````
</CodeGroup>

<ParamField path="name" param-type="string" required="true">
  Space display name.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained from dashboard.
</ParamField>
