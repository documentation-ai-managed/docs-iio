## Migration Guide

Upgrade iio smoothly with these steps. Review changes before updating.

<Callout kind="alert">
  Backup your docs before migrating from v1.0.0 or earlier.
</Callout>

<Steps>
  <Step title="Check Compatibility" icon="check-circle">
    Verify your config uses YAML format.
  </Step>
  <Step title="Update Package" icon="download">
    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install iio@latest
    ````
    ````bash
    yarn add iio@latest
    ````
    </CodeGroup>
  </Step>
  <Step title="Migrate Config" icon="settings">
    Convert JSON to YAML:

    ````yaml
    title: "Your Docs"
    description: "Updated description"
    theme:
      primary: "#3B82F6"
    ````
  </Step>
  <Step title="Test Changes" icon="play">
    Run `iio build` and check for errors.
  </Step>
</Steps>

<Expandable title="Advanced Migration" default-open="false">
  For API users, update endpoints:

  <CodeGroup tabs="JavaScript,Python">
  ````javascript
  const response = await fetch('https://api.example.com/v2/docs', {
    headers: { Authorization: `Bearer ${YOUR_TOKEN}` }
  });
  ````
  ````python
  import requests
  response = requests.get('https://api.example.com/v2/docs',
    headers={'Authorization': f'Bearer {YOUR_TOKEN}'})
  ````
  </CodeGroup>
</Expandable>
